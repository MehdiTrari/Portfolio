# Portfolio Mehdi Trari

Portfolio personnel de Mehdi Trari, developpeur full-stack oriente back-end PHP/Symfony avec une sensibilite AppSec.

Le site presente le profil, les competences, une selection de projets et les liens de contact.

## Stack

- Next.js 16
- React 19
- TypeScript
- Motion
- Radix UI Icons
- Docker

## Lancer le projet en local

```bash
npm install
npm run dev
```

Le site est ensuite disponible sur `http://localhost:3000`.

## Scripts utiles

```bash
npm run dev
npm run build
npm run start
npm run typecheck
```

## Docker

```bash
docker compose up --build
```

## Structure

- `app/page.tsx` : page principale du portfolio
- `app/layout.tsx` : metadata et layout racine
- `app/globals.css` : styles globaux et responsive
- `public/` : assets publics
